# Lead Automation Project

This project automates converting unstructured text input into structured CRM data using Google Sheets and Apps Script.

## Features
- Extracts Budget, Location, Requirement
- Automates lead processing
- Reduces manual effort

## Tech Used
- Google Sheets
- Google Apps Script
- JavaScript

## Example Input
"Wants 2bhk in gurgaon under 50 lakh urgent"

## Output
Budget: 50 lakh  
Location: gurgaon  
Requirement: 2BHK  
Next Step: Follow-up call
